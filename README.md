# Mother of God Meme Maker

Static React + Vite app. Host it for free on Vercel, Netlify, or GitHub Pages.

## Dev
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy (Vercel)
- Import this folder as a new project.
- Build: `npm run build`
- Output: `dist`

## Deploy (Netlify)
- Build: `npm run build`
- Publish: `dist`

## Deploy (GitHub Pages)
```bash
npm i -D gh-pages
# then push your repo and:
npm run build
npx gh-pages -d dist
```
